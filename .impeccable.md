# IMPECCABLE — EVA Design System

> Дизайн-система проекта EVA
> Версия: 1.0 | Дата: 08.04.2026
> Стиль: Минимализм + Адаптивная тема + Эмпатичный тон

---

## 🎨 Философия дизайна

**Минимализм.** Каждый элемент на экране несёт смысл. Никакого визуального шума. Пользователь в состоянии напряжения — интерфейс не должен давить.

**Адаптивность.** Уважение к выбору пользователя: светлая тема днём, тёмная ночью. Автоматическое определение системной темы.

**Эмпатия.** Тон — как разговор с тёплым, понимающим психологом. Никакого осуждения. Результаты подаются как открытие, не как диагноз.

## 🔤 Типографика

**Шрифт:** Inter (Google Fonts) — нейтральный, читаемый, профессиональный.

```css
/* Подключение */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap');

/* Иерархия */
--font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;

h1: 28px / 700   — заголовок экрана (результат, приветствие)
h2: 22px / 600   — заголовок секции (вопрос теста)
h3: 18px / 600   — подзаголовок (название шкалы)
body: 16px / 400 — основной текст, вопросы
small: 14px / 400 — подписи, метки
caption: 12px / 400 — вспомогательный текст
```

**Особенности:**
- Line-height: 1.5 для body, 1.3 для заголовков
- Letter-spacing: -0.02em для h1-h2 (плотнее, премиальнее)
- Максимальная ширина текста: 600px (читабельность)

## 🎭 Анимации (Framer Motion)

**Принцип:** Плавные, но не навязчивые. 200-300ms duration. Ease-out.

### Переходы между вопросами
```ts
// Появление вопроса
initial: { opacity: 0, y: 20 }
animate: { opacity: 1, y: 0 }
exit:    { opacity: 0, y: -20 }
transition: { duration: 0.25, ease: "easeOut" }
```

### Появление результата
```ts
// Шкалы "вырастают" снизу
initial: { scaleY: 0 }
animate: { scaleY: 1 }
transition: { duration: 0.5, ease: "easeOut", stagger: 0.1 }
```

### Микро-анимации
- Нажатие кнопки: `scale: 0.97` → `scale: 1.0`
- Появление карточки: `opacity: 0` → `opacity: 1`, `y: 16` → `y: 0`
- Прогресс-бар: плавное заполнение, `transition: width 0.3s ease`
- Реферальный счётчик: `count.up` анимация

## 💬 Тон коммуникации (Copywriting)

### Приветствие
> «Добро пожаловать в EVA. Этот тест поможет увидеть то, что обычно скрыто — вашу опору, на которую вы опираетесь в сложных ситуациях. Ответьте честно — здесь нет правильных или неправильных ответов.»

### Во время теста
> «Вы на полпути. Ещё несколько вопросов — и вы увидите свой результат.»

### Результат
> «Ваша доминирующая опора — **[Название]**. Это значит, что в сложных ситуациях вы склонны опираться на **[описание]**. Иногда эта опора искажается — и тогда она мешает расти. Ниже вы увидите, как это проявляется.»

### Блокировка (2 месяца)
> «Вы уже проходили тест. Чтобы результат оставался честным, повторное прохождение доступно через 2 месяца. Но вы можете пройти квалификационный опрос — он поможет определить, что для вас важно прямо сейчас.»

### Реферальная система
> «Хотите увидеть полную картину? Пригласите 2 друзей — и вы увидите свою вторичную опору. Это не просто добавка — это ключ к пониманию того, почему доминантная опора иногда даёт сбой.»

## 📱 Экраны приложения

1. **Welcome** — Приветствие + кнопка «Начать тест»
2. **Test** — 25 вопросов с прогресс-баром (один вопрос на экран)
3. **Loading** — Анимация «анализа» (2-3 сек, WOW-эффект)
4. **Result Main** — Доминантная опора + визуализация 5 шкал
5. **Referral Gate** — «Пригласите 2 друзей для вторичной опоры»
6. **Result Secondary** — Вторичная опора (после рефералов)
7. **Qualification** — 3 вопроса (сфера, уровень, попытки)
8. **Products** — Выбор продукта + ссылка в ЛС автора
