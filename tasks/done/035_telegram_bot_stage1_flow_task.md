<task>
1. В обработчике вебхуков Telegram (файл `app/api/webhook/telegram/route.ts` или аналогичный) переписать логику для команды `/start`.
2. При `/start` бот отправляет сообщение:
   - Фото: любое из папки `public` (указать корректный URL до картинки на сервере).
   - Текст: "Привет! Перед тем как начать, подпишись на канал. Там я даю информацию, которую не даю больше нигде."
   - Инлайн-клавиатура (НЕ Reply!):
     * Кнопка 1: "Подписаться" (url: `process.env.TELEGRAM_CHANNEL_URL`)
     * Кнопка 2: "Я подписалась" (callback_data: `check_subscription`)
3. Реализовать обработку `callback_query` с датой `check_subscription`:
   - Вызвать метод Telegram API `getChatMember` для `process.env.TELEGRAM_CHANNEL_ID` и ID пользователя.
   - Если статус `member`, `administrator` или `creator` -> отправить сообщение: "У каждого человека есть внутренняя «опора»... (взять текст из Этапа 3 ТЗ)" с кнопкой Web App "Посмотреть" (ведущей в Mini App).
   - Если статус другой -> показать уведомление (answerCallbackQuery), что подписка не найдена.
4. Перенести этот файл из tasks/todo/ в tasks/done/ после завершения.
