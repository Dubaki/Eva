<context>
Код отправлен на GitHub. Заказчик готовится к деплою на Vercel. Поскольку Заказчик использует Windows PowerShell, стандартная команда `curl` для установки Telegram Webhook может выдать ошибку парсинга JSON. Нужен надежный Node.js скрипт для регистрации вебхука.
</context>

<task>
1. Создать папку `scripts/` в корне проекта.
2. Написать скрипт `scripts/set-webhook.js`:
   - Скрипт должен читать `BOT_TOKEN` и `TELEGRAM_WEBHOOK_SECRET` из файла `.env.local` (использовать `dotenv`).
   - Скрипт должен запрашивать у пользователя Vercel URL (через аргументы командной строки `process.argv`).
   - Скрипт делает POST-запрос к API Telegram (`setWebhook`) с правильными заголовками и JSON-телом.
   - Выводит красивый лог успеха или ошибку.
3. Добавить в `package.json` команду: `"webhook": "node scripts/set-webhook.js"`.
4. Написать в лог для Заказчика короткую инструкцию:
   - Как задеплоить проект через панель Vercel.
   - Как запустить созданный скрипт (`npm run webhook https://твой-домен.vercel.app`).
5. Перенести этот файл в tasks/done/ после завершения.
</task>

<rules>
- Скрипт должен быть пуленепробиваемым (обрабатывать ошибки сети и ответы от Telegram).
- Использовать встроенный `fetch` Node.js (v18+).
