Админ-панель видна, но при попытке входа срабатывает редирект на начало теста. Нужно перенести вход в админку на стартовый экран и отключить принудительные перенаправления для тестировщиков.
</context>

<task>
**ЧАСТЬ 1: Перенос кнопки (Entry Point)**
1. Найти кнопку "👑 Админ-панель" и перенести её из компонента результатов на самый ПЕРВЫЙ экран приложения (Стартовая страница до начала теста).
2. Условие отображения остается прежним: 
   `const TESTER_IDS = [1149371967, 5930269100, 1419397753];`
   `const isVIP = TESTER_IDS.includes(Number(user?.tg_id));`
3. Кнопка должна быть аккуратной, располагаться либо в самом верху, либо внизу под основной кнопкой "Пройти тест".

**ЧАСТЬ 2: Отключение принудительного редиректа (Admin Bypass)**
4. Найти в коде логику, которая делает `router.push('/')` или редиректит пользователя, если у него `!dominant_trait`.
5. Добавить в это условие исключение: 
   "ЕСЛИ пользователь НЕ в списке TESTER_IDS И у него нет результата — ТОГДА редирект".
   Если пользователь — Тестировщик, ему ДОЛЖНЫ быть доступны любые роуты (включая `/admin`) вне зависимости от прохождения теста.

**ЧАСТЬ 3: Фикс сравнения ID**
6. Для надежности во всех проверках использовать приведение к строке:
   `const isVIP = TESTER_IDS.map(String).includes(String(user?.tg_id));`

7. Сделать git push и перенести в done/.
</task>
